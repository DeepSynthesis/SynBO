Overview
--------

ReactionOpt is designed to help chemists optimize reaction conditions efficiently using machine learning techniques. The package leverages Bayesian Optimization to intelligently explore the reaction parameter space and find optimal conditions for multiple objectives simultaneously.

Key Features
------------

* Multi-objective optimization (yield and enantioselectivity)
* Bayesian Optimization with Gaussian Process surrogate models
* GPU acceleration for large-scale problems
* Flexible parameter space definition
* Automated visualization of results
* Integration with high-throughput experimentation workflows
